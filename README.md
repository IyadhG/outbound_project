# Nudge AI - Autonomous Outbound Sales Agent

  

## Overview : This project was developed as part of the coursework at Esprit School of Engineering. It focuses on solving the B2B pipeline gap through autonomous AI agents and relationship mapping.  

## Features :  

Autonomous lead discovery and enrichment (Inject/Collect).  

Multi-agent reasoning with the "AI Jury" (Detective).  

Hyper-personalized messaging via Tone Cloning (Writer).  

## Tech Stack :  

Backend: Python, LangGraph, FastAPI.  

Frontend: React.js, TypeScript, Tailwind CSS.  

Databases: Neo4j (Graph), PostgreSQL (Relational).  

Other Tools: Docker, GitHub Education hosting.
